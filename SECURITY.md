# Security Policy

ReadMyPaper is a local-first beta app that handles uploaded PDFs, stores
generated artifacts on disk, serves a local web UI, downloads TTS model assets,
and can optionally send selected text snippets to an OpenAI-compatible LLM
endpoint.

Please do not open a public issue for security-sensitive findings.

Use GitHub Security Advisories through GitHub's private vulnerability reporting
flow instead:

```text
https://github.com/ThalesMMS/ReadMyPaper/security/advisories/new
```

Relevant security reports include issues around uploaded PDF handling, access to
stored PDFs or generated artifacts, unintended local web exposure, unsafe model
or voice downloads, and optional LLM endpoint handling.

This is a small volunteer-maintained project. You should expect an
acknowledgment within a reasonable timeframe, but there is no formal response
SLA.
